# Mis elecciones semanticas.

## Estructura semantica
use header, main, aside y footer para la tarjeta

- header: Lo use para el encabezado, en este caso, Mi nommbre y ademas use un nav para diferenciar los puntos

- main: ahi coloque la informacion principal, es la seccion con mas texto y lleva la información sobre mi.

- aside: ahi añadi una lista desorganizada (ul), en el contexto de la tarjeta, ahi puse mis habilidades y conocimientos

- footer: en este caso lo use para colocar la fecha de creación y quien lo hizo
